# Mithril-Webpack-Typescript-Boilerplate

## Features
- Mithril 1.11
- Webpack 3.0 +
- Typescript 2.4 + with tslint
- SASS with PostCSS autoprefixer

## Installation
```npm install```

## Development
```npm run start```

## Build
```npm run build```

Compiles minified js and css to the `output` directory.
